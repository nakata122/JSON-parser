/*! \mainpage JSON parser
 *
 * \section intro_sec Увод
 *
 * Проекта реализира клас JsonParser, който парсва файлове от формат Json и ги обработва с определени методи.
 *
 * \section install_sec Тестове и примери
 *
 * \subsection step1 Отваряне на Json file
 *  open json/test1.json \n
 *  Отваря примерен Json файл, който съдържа различни типове данни, като например string, int, object, array \n
 *  Възможно е наслагване на object в object или array в object  \n
 *  Извежда грешка, ако е възникнал проблем при парсването
 *  
 * \subsection step2 Validate на Json file
 *  validate \n
 *  Проверява, дали има правилно зареден файл в паметта 
 * 
 * 
 * \subsection step3 print
 *  print \n
 *  Извежда на конзолата заредения файл, ако е валиден
 * 
 * 
 * \subsection step4 Търсене по ключ
 *  search type \n
 *  Извежда на конзолата всички срещания на обекти със същия ключ. Не извежда нищо, ако няма такива
 * 
 * 
 * \subsection step5 Промяна на данните по даден път
 *  set Objects.computer.type "My new type" \n
 *  Променя данните на пътя, ако има такива
 * 
 * 
 * \subsection step6 Местене на данни от един път до друг път
 *  move Objects.computer.type Objects.phone.number \n
 *  Взема данните от първият път и ги изтрива от йерархията(само референцията към пътя, не самия път), след това се преместват във втория път, като той се изтрива напълно
 * 
 * \subsection step7 Създаване на нов път
 *  create a.b.c.d.e.f.g.p { "MyObject" : "empty" } \n
 *  Ако вече не съществува такъв път се прави нов път, по който да се достъпи и се сменя стойността спрямо типа данни въведен от потребителя. Той може да е произволен. Например тук съм направил нов обект
 * 
 * \subsection step8 Изтриване на път
 *  Изтрива данните от пътя, ако той съществува. 
 * 
 * \subsection step9 Затваряне на Json file
 *  close \n
 *  Изтрива заредения в момента файл от паметта. Самият файл, от който е прочетен не се използва изобщо
 * 
 * \subsection step10 Съхраняване
 *  save \n
 *  Съхранява целия отворен обект в същото име, с което е отворен файла \n
 *  save Objects \n
 *  Съхранява само обекта от подадения път нататък \n
 *  saveas new.json \n
 *  Съхранява целият отворен обект в файла new.json. Параметъра път създава и нов файл, ако не съществува \n
 *  saveas new.json Objects \n
 *  Съхранява само обекта от подадения път нататък в файла new.json
 */
